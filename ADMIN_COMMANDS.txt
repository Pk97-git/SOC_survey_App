=== ADMIN COMMANDS FOR YOU TO RUN ===

1. Open PowerShell as Administrator, then run:

# Create IIS Website for Frontend
Import-Module WebAdministration
New-Website -Name 'FacilityApp' -PhysicalPath 'C:\WebApps\facility-app' -Port 8080

# Alternative: Add to existing Default Web Site as virtual directory
# New-WebVirtualDirectory -Site 'Default Web Site' -Name 'app' -PhysicalPath 'C:\WebApps\facility-app'

2. Open Windows Firewall for port 8080:
netsh advfirewall firewall add rule name="FacilityApp Web" dir=in action=allow protocol=tcp localport=8080

3. Open Azure Portal and add NSG inbound rule for port 8080

=== END OF ADMIN COMMANDS ===
